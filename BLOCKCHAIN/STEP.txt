Setup needed:

set up truffle
npm install -g truffle

setup ganache 
connect truffle.js 
setup port 
setup compiler to version 0.8.13



Connect to the Metamask
https://github.com/MetaMask/legacy-web3
npm install @metamask/legacy-web3


Procedure

if in another nodes/file/pc double check the contract address -if you are using local
install truffle -npm install -g truffle-  if the new folder or project dont have truffle yet
add truffle.js directory in ganache if new file 
then truffle migrate and get the new contract address of smart contract
then update the contract address to your js setup
